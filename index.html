XHTML / XML

The DTD: Since XHTML is XML, it must be validated via a Document Type Definition.

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN" "...">

Heading: The <h1> must be lowercase and closed: <h1>Header</h1>.

HTML 5.0 Modern and updated.

The DOCTYPE: No DTD is required. It is a living standard.

<!DOCTYPE html>

The <h1> Tag: Supports multiple instances per page within semantic <article> or <section> tags.

The comments in html follow the syntax: <!-- comment content here --> Why use comments? You can use them as notes for functions like:

<!-- DOCTYPE -->

<!DOCTYPE html >




HTML 4.1 Legacy

The DTD: Based on SGML. Requires Strict, Transitional, or Frameset declarations.

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN">

The <h1> Tag: Traditionally one per page, often using align="center" attributes directly.
